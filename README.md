Java Swing Calculator

A desktop calculator application built using Java Swing, featuring real-time calculation capabilities via the Math.js API. This project was built to explore Java GUI development, event-driven programming, and asynchronous networking.
Features

    Intuitive GUI: Built using Java Swing with customizable themes (Blue, White, Gray).

    Cloud-Powered: Performs calculations using an external REST API to handle complex expressions.

    Responsive Design: Supports basic arithmetic (+, -, *, /) and square root operations.

    User Settings: Allows users to modify the UI theme through a settings interface.

Tech Stack

    Language: Java

    GUI Framework: Java Swing

    Networking: Java HttpClient (for API requests)

    Architecture: Event-Driven (ActionListener implementation)

Prerequisites

    Java Development Kit (JDK) 11 or higher (Required for HttpClient)
